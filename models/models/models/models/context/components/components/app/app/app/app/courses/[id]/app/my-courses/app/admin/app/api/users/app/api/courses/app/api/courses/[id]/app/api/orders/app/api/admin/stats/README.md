# MYFOOD ACADEMY

Online cooking course platform with Google login, Razorpay payments, and admin panel.

## Setup

1. Install: `npm install`
2. Add credentials to `.env.local`
3. Run: `npm run dev`

## Deploy on Vercel

Push to GitHub, import on Vercel, add env variables, deploy!

## Features

- Google Authentication
- Course purchase with Razorpay
- Admin dashboard
- My Courses section
- Responsive design

## Tech Stack

Next.js 14, MongoDB, Firebase Auth, Razorpay, Tailwind CSS
